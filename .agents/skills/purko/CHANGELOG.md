# Purko Skill — Changelog

## 1.0.0 — 2026-07-12
First versioned release of the Purko skill.

- Mission Control webapp: watch workflows/agents live, approve/decline human
  gates, workflow attention-sorting (running/gate first).
- Guided flows: create/manage agents (H), workflows (I), edit/delete (J) —
  cluster mode via kubectl, remote/hosted mode via the dashboard API.
- Flow G: connect to a hosted/ingress workspace by URL + token (or PAT).
- Onboarding: demo studio (5 industry showcases), self-host install guide,
  get-access paths.
- Interim self-version check: this masthead chip and the /purko notice tell
  you when a newer skill is published (until the plugin marketplace lands).
